### Áreas de Estudo e Prática

#### Engenharia de Dados (Ingestão & Armazenamento)
- **Pipelines:** Desenvolvimento de pipelines de ETL/ELT locais focados em integridade transacional e eficiência sob a arquitetura Medallion (camadas 🟤 Bronze, ⚪ Silver e 🟡 Gold).
- **Orquestração:** Gerenciamento, monitoramento e fluxo de tarefas utilizando Prefect Cloud.
- **Lakehouse / Motores:** Armazenamento colunar moderno e análise analítica veloz utilizando Polars, DuckDB, Parquet e Delta Lake.
- **Modelagem:** Estruturação de camadas analíticas e Data Marts organizados via DBT (Data Build Tool) local.

#### Ciência de Dados & Machine Learning
- **Arquiteturas:** Aprendizado de representações, redes neurais profundas (PyTorch) e Aprendizado por Reforço (PPO).
- **Validação:** Validação rigorosa out-of-sample (Simulações de Monte Carlo), busca de hiperparâmetros (Optuna) e calibração estocástica de incerteza para combater overfitting.
- **Analytics Estatístico:** Análise de séries temporais, modelos de contagem (zero-inflated) e distribuições probabilísticas complexas.

#### MLOps & SRE (Operação de Plataformas)
- **Automação:** Gerenciamento eficiente de ambientes virtuais locais utilizando `uv` (resolução ultra-rápida de dependências).
- **Observabilidade:** Criação de consoles de log estruturados, telemetria física em tempo real (monitoramento de CPU/RAM) e controle de travas físicas (`.lock`).
- **Deploy:** Modelagem otimizada para execução local (Air-Gapped) e exportação de modelos para o padrão ONNX.
